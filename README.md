# 📚 Book Seller - Full Stack Web Application

A full-stack web application designed to manage books, users, and orders efficiently. This project demonstrates end-to-end development including frontend, backend, and database integration.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 📖 Book Management (Add, Update, Delete)
- 🛒 Order Management System
- 👨‍💼 Admin Panel for managing users and books
- 📊 Responsive UI

---

## 🛠 Tech Stack

**Frontend:**
- HTML5
- CSS3
- JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
